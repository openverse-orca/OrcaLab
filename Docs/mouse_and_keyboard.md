# OrcaLab鼠标和键盘操作

> 如果焦点在文本框上（输入状态），所有快捷键无效。

## 切换平移、旋转和缩放

功能：旋转
按键：1
作用域：全局

功能：旋转
按键：2
作用域：全局


功能：旋转
按键：3
作用域：全局

> 按住Shift可以切换为世界坐标空间。


## 相机导航

功能：聚焦到选择物体
按键：z
作用域：全局

功能：相机向前移动
按键：按住鼠标右键 + w
作用域：视窗

功能：相机向后移动
按键：按住鼠标右键 + s
作用域：视窗

功能：相机向左移动
按键：按住鼠标右键 + a
作用域：视窗

功能：相机向右移动
按键：按住鼠标右键 + d
作用域：视窗

功能：相机向上移动
按键：按住鼠标右键 + e
作用域：视窗

功能：相机向下移动
按键：按住鼠标右键 + q
作用域：视窗

功能：转动相机
按键：按住鼠标右键 + 移动鼠标
作用域：视窗

功能：绕一点转动相机
按键：Alt + 按住鼠标做键 + 移动鼠标
作用域：视窗


## 仿真控制

功能：切换选择的物体控制状态（仿真控制/用户控制）
按键：F3
作用域：全局


功能：显示/隐藏物理碰撞和Joint
按键：F4
作用域：全局

## 编辑

功能：删除选择物体
按键：Delete
作用域：全局
